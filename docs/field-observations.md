# 現場觀察與模型失效案例

這些資料是匿名現場觀察或口述經驗，不是醫管局病人級數據，也不直接用作模型標籤。用途是找出錯誤假設、設計輸入欄位和提出可驗證假設。

## 2026-07：院內倒數失效

- 北區醫院傍晚：較少危急個案時，實際等待與官方約 3.5 小時估算接近。
- 威爾斯親王醫院 Cat IV：已等待約 12 小時，同級隊列約第二位，但較高優先度個案持續獲優先處理，舊版以官方時間減已等待時間後錯誤顯示應已完成。
- 結論：官方數字描述醫院／分流層級，不是個人服務完成時間；同級位置也不是整體隊列位置。院內模式不得以「官方估算 − 已等待」倒數。

## 容量與需求的早晨假設

- 有經驗醫生口述夜間人手較少。
- 現場經驗指約 07:00 人手和處理效率較高。
- 另一個同時存在的需求假設是：工作人口在 09:00 前自行到急症室或陪同家人，令到達量增加。
- 公開等候曲線只顯示到達和完成的淨結果。凌晨等候下降可能來自到達減少，不能視為效率提高；07:00 後曲線持平也不能否定容量增加。

驗證以上假設需要逐院、逐 15 分鐘、按分流級別的到達數、首次見醫生數／完成數，以及逐更臨床人手。現有公開 API 沒有這些欄位，因此效應量保持未知。
